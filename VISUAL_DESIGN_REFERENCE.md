## Component Styling Details

### Entity Details Card
- **Container**: White card with light border (border-slate-200)
- **Header**: Padding pb-4, title in slate-900
- **Grid**: 2 columns with gap-6
- **Labels**: Text-xs, font-medium, uppercase, slate-500 (muted)
- **Values**: Text-sm, font-medium, slate-900 (dark)

### Sync Status Table
- **Container**: White card with light border
- **Header**: Background slate-50, non-hoverable
- **Table Cells**: py-3 padding, text-sm, slate-700
- **Revision ID**: Badge with secondary styling, font-mono
- **Buttons**: Outline variant, size-sm, gap-2, h-8
- **Row Hover**: bg-slate-50 on hover, border-slate-100

### Badges Styling
- **System**: `variant="outline"` with `bg-slate-50`
- **Entity Type**: `variant="secondary"` with default slate background
- **Revision Count**: `variant="outline"` with `bg-blue-50 text-blue-700 border-blue-200`

### Icons Used
- **View XML**: `Eye` icon from lucide-react (h-3.5, w-3.5)
- **File XML**: `FileText` icon in stat widget
- **Modal Close**: Built-in dialog close button

## Responsive Behavior

### Desktop (lg screens and above)
- 2-column grid for entity details (side-by-side)
- Full-width table with all columns visible
- Modal at max-w-2xl
- Normal padding and spacing

### Tablet (md screens)
- 2-column grid may stack or adjust
- Table remains scrollable if needed
- Modal responsive

### Mobile (sm screens)
- Grid may convert to single column
- Table with horizontal scroll
- Modal full-screen or near full-screen
- Compact button sizing (h-8)
